# The Aqua Guide

## Getting Started
To run this project locally, follow these steps:

1. **Clone the repository:**
   `git clone https://github.com/nikhileshreddyb/tag.git`

2. **Setup Backend:**
   - `cd backend`
   - `npm install`
   - Create a `.env` file (ask admin for credentials)
   - `npm start`

3. **Setup Frontend:**
   - `cd ../frontend`
   - `npm install`
   - `npm run dev` (or `npm start`)
